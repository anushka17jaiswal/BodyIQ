# 🧠 BodyIQ AI

BodyIQ AI is an intelligent health analysis platform that helps users understand their health status through AI-powered insights, personalized recommendations, and interactive health assistance.

## 🚀 Features

### AI Health Analysis

* BMI Calculation
* Health Score Generation
* Health Risk Assessment
* Personalized Recommendations

### AI Health Assistant

* Real-time AI Chat
* Fitness Guidance
* Nutrition Recommendations
* General Health Queries

### Personalized AI Plans

* Workout Plans
* Diet Plans
* Sleep Recommendations

### Modern User Experience

* Responsive UI
* Voice Input Support
* AI Voice Responses
* Interactive Dashboard

## 📦 Installation

### Clone Repository

git clone https://github.com/anushka17jaiswal/BodyIQ.git

### Frontend

cd bodyiq-frontend

npm install

npm run dev

### Backend

cd backend

pip install -r requirements.txt

uvicorn main:app --reload
